1) Getting started

Bootstrap funciona con enfoque mobile first. Eso significa que diseñas primero para pantallas pequeñas y luego escalas a tablet y escritorio con breakpoints. Para iniciar solo necesitas doctype HTML5, meta viewport y los enlaces CDN. 

<!doctype html>
<meta name="viewport" content="width=device-width, initial-scale=1" />
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet" />
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>

2) Layout

El layout de Bootstrap se basa en contenedores, filas y columnas. Un patrón muy usado es col-12 col-md-6 col-xl-4: una columna en móvil, dos en tablet y tres en escritorio. Así logras un diseño responsivo sin escribir media queries desde cero.

col-12
col-md-6
col-xl-4
<div class="container">
  <div class="row g-3">
    <div class="col-12 col-md-6 col-xl-4">Bloque A</div>
    <div class="col-12 col-md-6 col-xl-4">Bloque B</div>
    <div class="col-12 col-md-6 col-xl-4">Bloque C</div>
  </div>
</div>

3) Content

Esta capa incluye tipografía, tablas, imágenes y el reset base (Reboot). En la práctica te ayuda a mantener consistencia visual entre navegadores y una lectura más limpia del contenido.

Area Clases clave
Tipografia display-*, lead, fw-*, text-*
Tablas table, table-striped, table-responsive 
<h2 class="display-6">Titulo</h2>
<p class="lead">Texto introductorio</p>
<table class="table table-striped">...</table>

Aquí puedes ver validación visual en tiempo real. Cuando escribes, cada campo cambia entre is-valid y is-invalid; al enviar, el formulario aplica was-validated y muestra feedback completo.

6) Helpers

Los helpers resuelven situaciones comunes sin escribir CSS extra: ratio para embeds responsivos, position para superposiciones, visually-hidden para accesibilidad y stretched-link para ampliar el área clickeable. 

<div class="ratio ratio-16x9">
  <iframe src="..." title="video"></iframe>
</div>

7) Utilities

Las utilities son el corazón de productividad: spacing, display, flex, text, color, border, shadow, rounded, opacity, overflow, sizing y z-index. Con ellas puedes construir interfaces completas con clases reutilizables. 

<div class="d-flex justify-content-between align-items-center p-3 rounded-3 shadow-sm bg-white">
  <span class="fw-semibold">Card compacta</span>
  <span class="badge text-bg-success">Activa</span>
</div>
