# 01 - HTML y CSS Básico

En esta parte se aprende a construir la estructura visual mínima de una página web.

## ¿Qué se aprende aquí?

1. Cómo se arma un archivo HTML desde cero.
2. Qué etiquetas se usan para organizar contenido.
3. Cómo aplicar estilos con CSS.
4. Cómo mantener el código ordenado y fácil de entender.

## HTML explicado de forma simple

HTML es el esqueleto de la página. Con él defines títulos, párrafos, imágenes, enlaces, listas y secciones.

## CSS explicado de forma simple

CSS es la parte visual. Con él decides colores, tamaños, separaciones, alineación y apariencia general.

## Temas principales

- Estructura mínima de un documento HTML.
- Etiquetas semánticas como `header`, `main`, `section`, `article` y `footer`.
- Texto, listas, imágenes y formularios básicos.
- Selectores, clases, márgenes, rellenos y colores en CSS.

## Qué debe poder hacer el alumno al terminar

- Crear una página simple por su cuenta.
- Entender la diferencia entre estructura y estilo.
- Saber dónde poner el contenido y dónde poner el diseño.
