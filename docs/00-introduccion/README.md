# 00 - Introducción

Esta es la primera parada del proyecto. Aquí se explica la idea general antes de ver código.

## ¿Qué debe entender una persona que empieza?

1. Qué es una página web.
2. Qué parte ve el usuario y qué parte trabaja detrás.
3. Cómo se relacionan HTML, CSS, JavaScript y Node.js.

## Explicación simple

- **Frontend**: es lo que el usuario ve y usa en el navegador.
- **Backend**: es lo que corre en el servidor y prepara datos o respuestas.
- **Conexión**: el frontend puede pedir información al backend cuando la necesita.

## Objetivo de esta sección

Después de leer esta carpeta, una persona sin experiencia debería poder responder:

- Qué hace el frontend.
- Qué hace el backend.
- Por qué una aplicación web necesita ambos.
